# AI Papers 全站紧凑密度修正

日期：2026-07-12  
状态：用户已确认视觉方向

## 目标

以 ob-ai-papers 与 ur-ai-papers 的实际页面密度为基准，收紧新站过大的行高、段间距、组件内边距和区块留白，同时保留三频道颜色、光谱线、统一论文流结构与全部现有功能。

## 调整范围

### 全局

- 正文基准由 15px / 1.6 调整为 14px / 1.5。
- 不改变 1320px 内容宽度，避免同时改变阅读列宽。
- 交互控件在手机端仍保持不低于约 40px 的可点击高度。

### 首页与导航

- 顶部导航高度由 72px 调整到约 56px。
- Hero 上下留白由 64–112px / 50px 调整到约 36–52px / 28px。
- Hero 标题最大字号由 72px 收到约 56px，标题与说明文字间距同步缩小。
- 频道按钮高度由 78px 调整到约 56–60px，内边距从 16×20px 调整到约 10×14px。
- 主内容区顶部间距由 46px 调整到约 24px，侧栏与结果区间距收至约 24px。

### 筛选与工具栏

- 筛选组上下内边距由 18px 调整到约 11–12px。
- 搜索和排序控件高度由 46px 调整到约 38–40px。
- 筛选 chip 高度由 28px 调整到约 24px，组内间距收至 4–5px。
- 结果标题上方间距由 34px 调整到约 18px。

### 论文卡

- 卡片间距由 12px 调整到约 8px。
- 卡片正文内边距由 21×23×19px 调整到约 13×15px。
- 中文标题调整到约 18–22px，行高约 1.3–1.35。
- 标题、英文标题、元数据、TL;DR、底部标签之间的垂直间距统一压缩。
- TL;DR 行高由 1.8 调整到约 1.55。
- 底部区域上边距与内边距由 17px / 13px 调整到约 10px / 8px。

### 详情、关于、空状态与页脚

- 详情页外层、标题区、频道摘要卡、原始摘要区整体收紧约 30%。
- 详情 TL;DR 与原始摘要行高控制在约 1.6。
- 关于页大段区块间距由 75–135px 级别调整到约 40–72px。
- 空状态、加载状态和页脚高度同步缩小，不保留另一套宽松节奏。

## 不改变

- 不改数据、筛选逻辑、路由、收藏、已读、RSS 和更新时间语义。
- 不改颜色、字体家族、三频道光谱线和领域轨道。
- 不删除 Hero、频道导航、论文摘要或标签。

## 验收

- 桌面首页一屏可看到更多论文卡，视觉密度接近旧站。
- 心理健康与用户研究频道保持同一节奏。
- 手机筛选抽屉没有过度拥挤，控件仍可点击。
- TypeScript 检查、Next.js 构建和现有浏览器冒烟测试通过。
- 更新桌面首页、心理健康频道与手机截图供验收。
